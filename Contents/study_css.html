CSSの機能一覧


ボーダーを設定する

ボーダーの色、線の種類、幅の長さを指定できる。

上下左右をそれぞれ設定もできる。

              .hoge{
                border-top: type width color;
                border-bottom: type width color;
                border-left: type width color;
                border-right: type width color;
              }

type : 線の種類

widht : 線の幅の長さ

color : 線の色


例： 
                hoge{
                  border-top: solid 10px red;
                  border-bottom: solid 10px blue;
                  border-left: solid 10px yellow;
                  border-right: solid 10px black;
                }

結果：

aaaa

ボーダーの線の種類を指定する

ボーダーの線の種類を指定できる

              hoge{
                border-style: type;
              }

上下左右それぞれ別にしてもできる

              hoge{
                border-top-style-style: type;
                border-bottom-style: type;
                border-right-style: type;
                border-left-style: type;
              }

type : 線の種類

type は下記の通り

dotted
dashed
solid
double
groove
ridge
inset
outset

上：dotted

下：dashed

左：solid

右：double


ボックスに影を付ける

ボックスに影を指定できる

              hoge{
                  box-shadow: none horizon vertical blur extent color inset;
              }

none:"none" と記述すると影を表示しない設定になる

horizon:影の表示する左右の座標 pxで指定する

負の値を指定することで左側に表示される

vertical:影の表示する上下の座標 pxで指定する

負の値を指定することで上側に表示される

blur:影のぼかし度

0 の値でぼかし無しで表示する。負の値は無効

extent:影の広がり具合

正の値で影が広がり、負の値で影が小さくなる

例：

              hoge{
                box-shadow: 6px 6px 10px 3px red;
              }

結果：

影のサンプル表示


テキストの行揃え

テキストを揃える指定ができる

              hoge{
                text-align: type;
              }

type : 揃える位置

type は下記通り

left : 左揃え

center : 中央揃え

right : 右揃え

他に justify,start,end がある模様

例：

              hoge1{
                text-align: left;
              }
              hoge2{
                text-align: center;
              }
              hoge3{
                text-align: right;
              }

結果：

左揃え
中央揃え
右揃え
